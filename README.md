# 📊 Customer Churn Analysis Dashboard (Power BI)

## 📌 Project Overview
This project focuses on creating an interactive dashboard using Power BI to analyze customer churn data. The dashboard helps identify patterns and insights that explain why customers leave a service.

The goal of this project is to demonstrate practical skills in data visualization and business intelligence using Power BI.

---

## 🛠 Tools & Technologies Used

- Power BI
- Data Visualization
- Data Cleaning
- Business Intelligence

---

## 📂 Dataset

Dataset used in this project:

churn-bigml-80.csv

The dataset contains information about customer usage patterns and whether the customer churned or not.

### Dataset Features

- State
- Account Length
- International Plan
- Voice Mail Plan
- Customer Service Calls
- Total Day Minutes
- Total Eve Minutes
- Total Night Minutes
- Charges
- Churn (Target Variable)

Total Records: 2666

---

## 📊 Dashboard Features

The Power BI dashboard includes the following visualizations:

### KPI Cards
- Total Customers
- Churn Customers

### Charts
- Pie Chart – Churn Distribution
- Column Chart – Customer Service Calls vs Churn
- Bar Chart – International Plan vs Churn
- Column Chart – Average Day Minutes vs Churn

### Interactive Filters
- State
- International Plan
- Voice Mail Plan

These filters allow users to explore churn patterns interactively.

---

## 📈 Key Insights

- Total Customers: 2666
- Churn Customers: 388
- Customers with more service calls are more likely to churn
- International plan customers show different churn behavior
